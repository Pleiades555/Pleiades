Maintenance tools

Security note: do not put your GitHub token inside this HTML file. This page only accepts a token at runtime and stores it in this browser only if you tick the save option. Use a fine-grained GitHub token limited to this repository/content access.
Adding PDFs normally
Upload current PDFs into Ford/TSB/pdf/. Upload known superseded PDFs into Ford/TSB/archive/superseded/.
Regenerating
After edits or uploads, run Update Ford TSB Index. The admin tools below can also request the workflow run automatically.
Manual corrections
Edits are saved to Ford/TSB/data/manual-overrides.json. The parser then merges those corrections into tsb-index.json.
Advanced passcode
Advanced view is hidden by default. Enter 11290 in the top-right passcode field.
Parts lists
Use the Parts Lists tab to add whole-bulletin parts or variant-specific parts. These are saved separately in Ford/TSB/data/variant-parts.json so the parser will not overwrite them.
Add parts that apply to the whole bulletin in the first box. Add variant-specific lists underneath. Format each line as part number | qty | description | notes. Example: AB39-12345-A | 1 | Harness kit | Use when connector damaged.

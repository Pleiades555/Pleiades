Maintenance instructions

Adding PDFs
Upload current PDFs into Ford/TSB/pdf/. Upload known superseded PDFs into Ford/TSB/archive/superseded/.
Regenerating the index
Go to GitHub Actions and run Update Ford TSB Index. The script writes Ford/TSB/data/tsb-index.json.
Manual corrections
Use Ford/TSB/data/manual-overrides.json if a PDF title, year, model or concern extracts incorrectly.
Passcode
Advanced view is hidden by default. Enter 11290 in the top-right passcode field to unlock this panel.

# Pleiades agent rules

- Preserve manual entries; never overwrite them during imports.
- Do not present guessed data as confirmed.
- Source important claims and keep source documents separate from structured facts.
- Distinguish families from exact codes for engines, transmissions and differentials.
- Imports must merge by stable IDs rather than wipe files.
- Retain legacy compatibility until an explicit cutover PR.
- Run V4 validation and regression tests before completing changes.
